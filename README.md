# FridgeFooty

A simple printable AFL game week sheet for kids and families.

This page shows the upcoming AFL round, lets you add player names, choose the sheet owner, and print a one-page tipping sheet for the fridge.

## Files

- `index.html`
- `Croc.jpg`
- `Koala.jpg`

## How to Use

1. Open the website in your browser.
2. Expand `Player Settings`.
3. Add each player name.
4. Choose the sheet owner.
5. Print the page in landscape.

## Printing

- Use landscape orientation.
- Use standard A4 paper.
- Print at 100% scale for best results.

## Notes

- The page automatically shows the current upcoming AFL round based on the date.
- Fixture data is embedded in the page.
- If AFL updates late-season fixture details, the HTML file will need to be updated and re-uploaded.
